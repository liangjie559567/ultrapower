## ultrapower v5.0.9

### 修复内容

- **HUD 标签英文化**：将 HUD 元素标签从中文恢复为英文，确保测试断言通过
  - 截断提示 `行` → `lines`，更新通知 `有更新` → `omc update`
  - `DEFAULT_HUD_CONFIG` 默认值修正：`maxOutputLines: 6` → `4`，`gitBranch/model` 默认关闭
  - `renderAgentsByFormat` multiline 分支修复
  - `context-warning`、`git`、`thinking`、`skills` 元素标签英文化

- **HUD autoCompact 无限循环**：修复 context-warning 在上下文恢复正常后仍持续触发 compact 的 bug

- **skills/hud MINGW64 兼容性**：修复 Git Bash 下 `!` 字符转义问题

- **5 个新专业 Agent**：`database-expert`、`devops-engineer`、`i18n-specialist`、`accessibility-auditor`、`api-designer`

- **MCP 集成扩展**：新增 Slack、Jira/Linear、PostgreSQL、Playwright、Sequential Thinking、Software Planning Tool
