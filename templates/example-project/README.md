# Ultrapower Example Project

这是一个简单的 TypeScript 示例项目，用于演示 Ultrapower 的基本使用。

## 快速开始

```bash
# 安装依赖
npm install

# 编译
npm run build

# 运行
npm start
```

## 项目结构

```
example-project/
├── src/
│   └── index.ts      # 主入口文件
├── package.json      # 项目配置
├── tsconfig.json     # TypeScript 配置
└── README.md         # 说明文档
```

## 使用 Ultrapower

在此项目中，你可以尝试：
- `/ultrapower:analyze` - 分析代码
- `/ultrapower:build-fix` - 修复构建错误
- `/ultrapower:code-review` - 代码审查
