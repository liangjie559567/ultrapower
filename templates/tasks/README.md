# 任务模板库

## 可用模板

### 功能开发

文件: `feature-development.md`
适用于: 实现新功能或改进现有功能
包含: 需求分析、设计、实现、测试、文档

### Bug 修复

文件: `bug-fix.md`
适用于: 修复 bug 和问题
包含: 问题复现、根因分析、修复、回归测试

### 代码审查

文件: `code-review.md`
适用于: 审查代码变更
包含: 质量检查、功能验证、性能安全、文档测试

### 重构

文件: `refactoring.md`
适用于: 改进代码结构
包含: 现状分析、设计、实现、测试、文档

### 安全审计

文件: `security-audit.md`
适用于: 安全审查和加固
包含: 威胁建模、代码审查、依赖检查、配置审查

## 使用方式

```bash

# 查看模板

cat templates/tasks/feature-development.md

# 应用模板到任务

/template apply feature-development "功能描述"
```

## 模板特点

* ✅ 预定义检查清单

* ✅ 标准化工作流程

* ✅ 最佳实践指导

* ✅ 易于定制扩展
