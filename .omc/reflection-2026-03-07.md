```

WAL 流程 = writeEntry → 状态写入 → commit → cleanup
```

### 2. 进程清理升级

```
SIGTERM (5s timeout) → SIGKILL (强制终止)
```

### 3. TOCTOU 消除

* 移除 `existsSync` check-then-use

* 直接异步操作 + 错误处理

### 4. 缓存一致性

* 异步读取后双重检查缓存

* 避免覆盖新鲜数据
