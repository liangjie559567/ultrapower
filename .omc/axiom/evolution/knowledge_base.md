- **适用场景**: 所有异步进程通信（stdin/stdout/stderr）

- **实现**:
  ```typescript
  const hasAgents = (input.agentsSpawned ?? 0) > 0;
  const hasModes = (input.modesUsed?.length ?? 0) > 0;
  ```
