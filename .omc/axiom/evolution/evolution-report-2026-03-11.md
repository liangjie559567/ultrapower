### 新增知识条目（4个）

#### 1. Claude Agent SDK Zod Schema 规范 [P0]
- **来源：** 修复 31 个 TypeScript 类型错误
- **核心发现：** SDK tool() 函数要求 Zod schemas，不接受 JSON Schema
- **转换规则：**
  - `{ type: "string" }` → `z.string()`
  - `{ type: "boolean" }` → `z.boolean().optional()`
  - `{ type: "array" }` → `z.array(z.string())`
  - 枚举约束 → `z.enum(['val1', 'val2'])`
- **应用范围：** 所有 MCP server 工具定义
- **置信度：** 高（已验证）

#### 2. 类型安全枚举参数设计 [P1]
