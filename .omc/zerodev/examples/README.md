# ZeroDev 示例

本目录包含 ZeroDev 的使用示例。

## 示例列表

### example1-workflow.ts
完整工作流演示：从需求识别到代码生成的完整流程。

```bash
npx ts-node examples/example1-workflow.ts
```

### example2-memory.ts
项目记忆集成：展示如何使用项目记忆推断平台类型。

```bash
npx ts-node examples/example2-memory.ts
```

### example3-errors.ts
错误处理：展示各种输入验证和错误处理场景。

```bash
npx ts-node examples/example3-errors.ts
```

### example4-templates.ts
所有模板：展示 5 种内置模板的使用。

```bash
npx ts-node examples/example4-templates.ts
```

### example5-concurrent.ts
并发调用：展示并发生成多个类的性能。

```bash
npx ts-node examples/example5-concurrent.ts
```

## 运行所有示例

```bash
npm run examples
```
