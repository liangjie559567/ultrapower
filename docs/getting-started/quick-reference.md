# 快速参考卡片

常用命令速查表。

---

## 🎯 执行模式

```bash
# 全自主执行（简单功能）
/ultrapower:autopilot "创建登录组件"

# 多 agent 协作（复杂任务）
/ultrapower:team "重构 API 层"

# 持续执行（长期任务）
/ultrapower:ralph "完成整个功能"

# 并行工作（最大并行度）
/ultrapower:ultrawork "并行处理多个模块"

# 战略规划
/ultrapower:plan "规划项目架构"
```

---

## 🔧 诊断与管理

```bash
# 诊断安装问题
/ultrapower:omc-doctor

# 查看执行历史
/ultrapower:trace

# 取消正在执行的任务
/ultrapower:cancel

# 查看系统状态
/ultrapower:ax-status
```

---

## 📚 Axiom 工作流

```bash
# 需求澄清
/ultrapower:ax-draft "创建用户认证系统"

# 专家评审
/ultrapower:ax-review

# 任务拆解
/ultrapower:ax-decompose

# 开始实施
/ultrapower:ax-implement

# 反思总结
/ultrapower:ax-reflect
```

---

## 🛠️ 开发工作流

```bash
# 代码审查
/ultrapower:code-review

# 安全审查
/ultrapower:security-review

# 测试驱动开发
/ultrapower:tdd "编写登录测试"

# 修复构建错误
/ultrapower:build-fix
```
