# 任务模板库用户指南

## 概述

任务模板库提供预定义的任务模板，帮助你快速启动常见工作流程。

## 可用模板

### 功能开发模板

```
/template feature-dev
```

包含：
- 需求分析
- 设计审查
- 实现
- 测试
- 文档更新

### Bug 修复模板

```
/template bug-fix
```

包含：
- 问题复现
- 根因分析
- 修复实现
- 回归测试
- 验证通过

### 代码重构模板

```
/template refactor
```

包含：
- 现状分析
- 重构设计
- 实现变更
- 测试验证
- 性能对比

### 性能优化模板

```
/template performance
```

包含：
- 性能基线测试
- 瓶颈识别
- 优化实现
- 性能验证
- 文档更新

### 文档编写模板

```
/template documentation
```

包含：
- 内容大纲
- 示例代码
- 最佳实践
- 常见问题
- 审查发布

## 使用模板

### 查看模板详情

```
/template list
```

显示所有可用模板及其描述。

### 应用模板

```
/template apply feature-dev "添加用户认证"
```

系统会：
1. 创建任务结构
2. 生成检查清单
3. 推荐执行模式
4. 设置时间估计

### 自定义模板

```
/template create my-template
```

创建自己的模板供后续使用。

## 最佳实践

1. **选择合适的模板**：根据任务类型选择
2. **自定义检查清单**：根据项目需求调整
3. **保存常用模板**：创建团队标准模板
4. **定期更新模板**：基于经验改进

## 常见问题

**Q: 如何修改现有模板？**
A: 使用 `/template edit template-name` 修改。

**Q: 如何删除模板？**
A: 使用 `/template delete template-name` 删除。

**Q: 模板可以共享吗？**
A: 可以，导出模板后分享给团队成员。
