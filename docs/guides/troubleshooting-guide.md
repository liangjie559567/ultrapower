# 故障排查手册

## 概述

本手册帮助你快速诊断和解决常见问题。

## 常见问题

### 向导相关

**问题：向导无法启动**

* 检查网络连接

* 清除浏览器缓存

* 重启应用

**问题：推荐结果不准确**

* 提供更详细的任务描述

* 检查历史数据是否正确

* 尝试手动选择模式

### 执行模式相关

**问题：任务执行失败**

* 查看错误日志：`/logs`

* 检查依赖是否安装

* 尝试使用 `executor` 模式

**问题：执行超时**

* 增加超时时间设置

* 分解为更小的任务

* 检查系统资源

### 工作流相关

**问题：工作流推荐不显示**

* 确保有足够的历史数据

* 检查推荐引擎是否启用

* 重启应用

**问题：模板无法应用**

* 验证模板是否存在

* 检查模板格式

* 尝试重新创建模板

## 诊断步骤

### 第 1 步：收集信息

```
/diagnose
```

系统会收集：

* 应用版本

* 系统信息

* 最近的错误日志

* 配置信息

### 第 2 步：查看日志

```
/logs --last 100
```

查看最近 100 条日志记录。

### 第 3 步：重置状态

```
/reset --safe
```

安全重置应用状态（保留数据）。

## 性能问题

**症状：应用响应缓慢**

检查清单：

* [ ] 关闭不必要的后台任务

* [ ] 清理临时文件

* [ ] 检查磁盘空间

* [ ] 重启应用

**症状：内存占用过高**

解决方案：

* 减少并发任务数

* 清理缓存

* 重启应用

## 数据恢复

**丢失任务数据**
```
/recover --type tasks
```

**恢复工作流历史**
```
/recover --type workflow
```

**完全恢复**
```
/recover --full
```
