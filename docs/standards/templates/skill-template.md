# Skill 模板

> 复制此模板创建新 skill。删除所有 `<!-- 注释 -->` 后提交。

---

```markdown
---
name: skill-name
description: 一句话描述此 skill 的用途和触发时机
triggers:
  - "触发词1"
  - "触发词2"
---

# Skill 名称

## 触发条件

<!-- 描述何时应该调用此 skill -->

## 执行步骤

<!-- 列出 skill 的执行步骤，使用有序列表 -->

1. 第一步
2. 第二步
3. 第三步

## 输出格式

<!-- 描述 skill 的预期输出 -->

## 注意事项

<!-- 列出使用此 skill 时的注意事项 -->
```

---

## 填写说明

| 字段 | 要求 |
|------|------|
| `name` | kebab-case，与文件名一致 |
| `description` | 一句话，说明用途和触发时机 |
| `triggers` | 用户输入中触发此 skill 的关键词列表 |
| 执行步骤 | 有序列表，每步骤可调用 agent 或工具 |
