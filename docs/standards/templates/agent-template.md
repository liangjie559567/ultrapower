# Agent 模板

> 复制此模板创建新 agent。删除所有 `<!-- 注释 -->` 后提交。

---

```markdown
# Agent 名称

<!-- 一句话描述此 agent 的职责 -->

## 角色定义

你是 ultrapower 的 [角色名称]，专注于 [核心职责]。

## 核心职责

<!-- 列出此 agent 的主要职责，3-5 条 -->

1. 职责一
2. 职责二
3. 职责三

## 工作流程

<!-- 描述 agent 的标准工作流程 -->

1. 接收任务
2. 分析输入
3. 执行核心逻辑
4. 输出结果

## 输出格式

<!-- 描述 agent 的标准输出格式 -->

## 约束条件

<!-- 列出 agent 不应做的事情 -->

- 不做 X
- 不做 Y
```

---

## 填写说明

| 字段 | 要求 |
|------|------|
| 角色定义 | 明确角色名称和核心职责 |
| 核心职责 | 3-5 条，具体可操作 |
| 工作流程 | 有序步骤，可引用其他 agent |
| 输出格式 | 明确输出结构，便于下游 agent 解析 |
| 约束条件 | 明确边界，防止越权操作 |

## 模型路由建议

| 复杂度 | 推荐模型 |
|--------|---------|
| 简单查找/扫描 | `haiku` |
| 标准实现/审查 | `sonnet` |
| 架构/深度分析 | `opus` |
